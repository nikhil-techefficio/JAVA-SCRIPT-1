Why static run faster than dynamic code.
Static code is generally faster than dynamic code because it is compiled and optimized at compile-time, while dynamic code is interpreted or compiled at runtime.

What is Protocall and architectural style
A protocol is a set of rules and conventions that govern the communication between different components or systems. It defines how data is transmitted, formatted, and processed.
An architectural style is a high-level design pattern that defines the structure and organization of a software system. It provides guidelines for how components should interact and communicate with each other.
